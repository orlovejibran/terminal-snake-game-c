## 🎮 About the Project

This is a simple Snake game built using pure C and running in the terminal.

You control a snake inside a box:
- Eat food (*) to grow
- Avoid hitting walls
- Avoid hitting yourself

---

## 🎮 Controls

- W → Move Up  
- S → Move Down  
- A → Move Left  
- D → Move Right  
- P → Pause / Resume  
- X → Quit Game  

---

## ⚙️ Game Features

- Multiple difficulty levels (Easy, Normal, Hard)
- Score tracking
- Snake grows after eating food
- Game over when hitting wall or itself
- Pause and resume anytime

---

## ▶️ How to Run

1. Open in Code::Blocks
2. Use MinGW compiler
3. Compile and Run
